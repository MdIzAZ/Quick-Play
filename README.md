# Quick Player

A video player app.

Features 

    1. Video play in both Foreground and Background.
    2. Picture-in-picture (PIP) mode.
    3. Video player speed control.
    4. Video Resize.
    5. Sorting videos (eg: Size, Name, Date etc).
